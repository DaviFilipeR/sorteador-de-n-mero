O projeto do sorteador de números é um projeto disponibilizado no curso de prática da lógica de programação da Alura que tem o intuito de fazer o aluno praticar suas habilidades com conceitos envolvendo sorteio de números. O projeto consiste em uma página onde o usuário digita uma quantidade de números a ser sorteada, determinando número mínimo e máximo e recebendo números aleatórios de dentro do intervalo determinado.
